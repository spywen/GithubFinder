This application is dedicated to the amazing Github service which exists since 2008. More and more developers and common people use this web service to share their projects and consequently their knowledge. GithubFinder application have been designed and created to be able to simply search and find what you want on this code encyclopaedia from a Windows device.

Thanks to the Github API ;)

Designed and created by Babin Laurent for school project at Supinfo Nice 2015 (4NET)

This application is devided in two well define parts :

  • A search page: to be able to enter keywords and get what have been found, inside a list of all the repositories clickable...
  • A details page: for a particular repository, which gathering all the more important information concerning this one (Owner data, repository data, search fragments found)

The details page contains as well many links which will redirect you on the Github web site.

Thanks to the application bar you will be able anytime, anywhere, to search for a new repository or share it!
